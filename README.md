Нужно разработать таблицу в формате Single Page Application.

**Требования к таблице.**

1. Таблица должна содержать 4 колонки:
    1. Дата
    2. Название
    3. Количество
    4. Расстояние
2. Таблица должна иметь сортировку по всем полям кроме даты. Фильтрация должна быть в виде двух выпадающих списков и текстового поля:
    1. Выбор колонки, по которой будет фильтрация
    2. Выбор условия (равно, содержить, больше, меньше)
    3. Поле для ввода значения для фильтрации
3. Таблица должна содержать пагинацию

Вся таблица должна работать без перезагрузки страницы.

**Можно использовать:**
- React/Axios
- css библиотеки

**Нельзя использовать:**

- Библиотеки с готовыми компонентами или плагины для React, которые предоставляют готовый функционал, требуемый в задании
- Библиотеки и плагины для валидации
